# მეუფის კონდაკი — მობილური წიგნი

მობილურზე მორგებული ვებ-აპლიკაცია (PWA) მღვდელმთავრის მსახურებების თვალყურის დევნებისთვის.

**ცოცხალი ვერსია: https://gulani.ge** (სარეზერვო: https://xokaido.github.io/meufis-kondaki/)

ჰოსტინგი: Cloudflare Workers (სტატიკური ფაილები `app/`-დან, კონფიგი —
`wrangler.jsonc`; ყოველი push-ისას Cloudflare თავად უშვებს `node build.js`-ს).

## გაშვება

```sh
cd app
python3 -m http.server 8742
```

გახსენით ტელეფონის/კომპიუტერის ბრაუზერში: `http://localhost:8742`
(ან ატვირთეთ `app/` საქაღალდე ნებისმიერ სტატიკურ ჰოსტინგზე — GitHub Pages, Netlify და ა.შ.)

## შესაძლებლობები

- სამი მსახურება საწყის ეკრანზე, წაკითხვის პროგრესით
- ფერებითა და ნიშნებით გამოყოფილი მთქმელები: მღვდელმთავარი (☩ ჯვარი), მღვდელი (ჯვარი), დიაკონი (საცეცხლური), გუნდი (ნოტები), მედავითნე/მუხლი (წიგნი)
- ლოცვების ჯგუფები (ლოცვა ა, ბ, გ…) — ჩაკეცვა/გაშლა სათაურზე შეხებით
- განგების მითითებები (რუბრიკები) — დახრილი, ჩაქრობადი ღილაკით
- სარჩევი ☰ — სწრაფი გადასვლა მსახურების ნაწილებზე
- შრიფტის ზომა A−/A+, ბნელი/ნათელი თემა
- „⏾ ეკრანი" — ეკრანი არ ჩაქრება მსახურების დროს (Wake Lock)
- პოზიცია ინახება — გახსნისას იქ დაბრუნდებით, სადაც გაჩერდით

## სტრუქტურა

- `*.md` — მსახურებების საწყისი ტექსტები
- `build.js` — გადამყვანი: markdown → `app/data/*.json` + `app/sw.js` (`node build.js`)
  - „პარაკლისი და ლოცვანი" ფაილი იყოფა შვიდ ნაწილად: კმევები, მცირე პარაკლისი, ლიტანიობა, ჯვართამაღლების ცისკარზე, ლოცვები ზიარების წინ, განსატევებელნი, მცირე კურთხევანი
- `convert_latin.py` — ერთჯერადი გადამყვანი AcadNusx-ის ლათინური კოდირებიდან ქართულზე (ორიგინალი შენახულია `*.latin-original.bak`-ში)
- `sw.template.js` — service worker-ის შაბლონი (ვერსიას და ფაილების სიას build.js ჩასვამს)
- `app/` — სტატიკური PWA (index.html, style.css, app.js, data/, fonts/, icons/, manifest, sw.js)

ტექსტის შესწორების შემდეგ ხელახლა გაუშვით `node build.js` — ის თავად
განაახლებს ქეშის ვერსიას, ასე რომ დაყენებული აპლიკაცია შემდეგ გახსნაზე
ახალ ტექსტს ჩამოტვირთავს.

## iOS-ზე დაყენება

გახსენით https://xokaido.github.io/meufis-kondaki/ Safari-ში →
გაზიარების ღილაკი → **Add to Home Screen**. აპლიკაცია სრულად მუშაობს
ინტერნეტის გარეშე.

## ტექსტში შეცდომის გასწორება (ბრაუზერიდან, დაყენების გარეშე)

1. გახსენით https://github.com/xokaido/meufis-kondaki
2. დააწკაპუნეთ სასურველ `.md` ფაილზე (მაგ. „მეუფის კონდაკი მწუხრი.md")
3. ზედა მარჯვნივ აირჩიეთ ფანქრის ✏️ ნიშანი (Edit this file)
4. გაასწორეთ ტექსტი და ზევით მარჯვნივ დააჭირეთ **Commit changes** → კიდევ ერთხელ **Commit changes**
5. დაახლოებით 1–2 წუთში საიტი თავად განახლდება; დაყენებული აპლიკაცია
   ახალ ტექსტს მომდევნო გახსნაზე ჩამოტვირთავს

ფაილების ფორმატი: **გამუქებული** მოკლე სტრიქონი = სათაური, გამუქებული
აბზაცი = ლოცვა, „გუნდი:", „დიაკონი:" და მისთ. = მთქმელი, ჩვეულებრივი
აბზაცი = განგების მითითება. ტექსტში სიტყვის გამოსაყოფად შემოსაზღვრეთ
ერთი ვარსკვლავით: `*შიოსათვის*` — გამოჩნდება გამუქებული, ოქროსფერი.

## განახლების გამოქვეყნება (ავტომატური)

ყოველი push `main`-ზე თავად აშენებს და აქვეყნებს საიტს
(GitHub Actions → `.github/workflows/deploy.yml`). ლოკალურად საკმარისია:

```sh
git add -A && git commit -m "..." && git push origin main
```
