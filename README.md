# მეუფის კონდაკი — მობილური წიგნი

მობილურზე მორგებული ვებ-აპლიკაცია მღვდელმთავრის მსახურებების (მწუხრი, ცისკარი, ლიტურგია) თვალყურის დევნებისთვის.

## გაშვება

```sh
cd app
python3 -m http.server 8742
```

გახსენით ტელეფონის/კომპიუტერის ბრაუზერში: `http://localhost:8742`
(ან ატვირთეთ `app/` საქაღალდე ნებისმიერ სტატიკურ ჰოსტინგზე — GitHub Pages, Netlify და ა.შ.)

## შესაძლებლობები

- სამი მსახურება საწყის ეკრანზე, წაკითხვის პროგრესით
- ფერებით გამოყოფილი მთქმელები: მღვდელმთავარი ☩, მღვდელი, დიაკონი, გუნდი, მუხლი
- განგების მითითებები (რუბრიკები) — დახრილი, ჩაქრობადი ღილაკით
- სარჩევი ☰ — სწრაფი გადასვლა მსახურების ნაწილებზე
- შრიფტის ზომა A−/A+, ბნელი/ნათელი თემა
- „⏾ ეკრანი" — ეკრანი არ ჩაქრება მსახურების დროს (Wake Lock)
- პოზიცია ინახება — გახსნისას იქ დაბრუნდებით, სადაც გაჩერდით

## სტრუქტურა

- `*.md` — მსახურებების საწყისი ტექსტები
- `build.js` — გადამყვანი: markdown → `app/data.js` (`node build.js`)
- `app/` — სტატიკური აპლიკაცია (index.html, style.css, app.js, data.js)

ტექსტის შესწორების შემდეგ ხელახლა გაუშვით `node build.js`.
